# ISP Provider - Сайт интернет-провайдера

Современный сайт интернет-провайдера, построенный на Astro с использованием Alpine.js для интерактивности.

## Функционал

- ✅ Главная страница с hero-секцией
- ✅ Калькулятор тарифов с интерактивным выбором скорости
- ✅ Карта покрытия с визуализацией районов
- ✅ Страница новостей
- ✅ Форма обратной связи
- ✅ Заказ обратного звонка
- ✅ Адаптивный дизайн для мобильных и десктопных устройств

## Технологии

- **Astro** - современный фреймворк для статических сайтов
- **Alpine.js** - легковесная библиотека для интерактивности
- **CSS** - чистый CSS с адаптивным дизайном

## Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск dev-сервера
npm run dev

# Сборка для продакшена
npm run build

# Просмотр собранного проекта
npm run preview
```

## Страницы

- `/` - Главная страница
- `/tariffs` - Тарифы и калькулятор

## Особенности

- Полностью адаптивный дизайн
- Интерактивные формы с валидацией

## Добавление новостей

Новости хранятся в формате Markdown в папке `src/content/news/`. Для добавления новой новости создайте файл `.md` с frontmatter:

```markdown
---
title: "Заголовок новости"
description: "Краткое описание"
pubDate: 2024-01-15
image: "/path/to/image.jpg"
tags: ["тег1", "тег2"]
---

Содержание новости в формате Markdown...
```
